# repofit v1.2.1

Supported-stack patch release. Node, Python, .NET, and Go are the primary
first-class stacks in the default toolchain model.

## Changes

- **Primary stack selection now follows runnable command coverage.** A Python,
  .NET, or Go repository with a small Node sidecar no longer gets treated as
  Node-primary just because a `package.json` exists.
- **Python toolchain inference is broader.** Repofit now recognizes common
  Python tool hints from `pyproject.toml`, `requirements*.txt`, and config files
  such as `pytest.ini`, `ruff.toml`, `mypy.ini`, and `pyrightconfig.json`.
- **Corpus probes are less Node-shaped.** Test safety, error actionability, and
  public-surface coverage probes now use the resolved toolchain and understand
  Python, .NET, and Go signals where applicable.

## Upgrading

```bash
npm install --save-dev @esbenwiberg/repofit@1.2.1 @esbenwiberg/corpus-default@1.2.1
```
